# RWFileTransferDemo

本Demo主要是把公司项目中用到的文件传输技术整理出来，其中使用到iOS 7.0 后退出的Mutipeer Connectivity框架，借助WIFI、蓝牙来建立传输通道，达到用户互相传输媒体的功能
